# Wedding Invitation Website

## How to Use
This website is designed for inviting guests to the wedding. To start using the website, simply open it in your web browser. You will see the details of the wedding event, along with options to RSVP.

## How to Add Photos
1. Navigate to the `photos` directory in the repository.
2. Add your desired photos in JPEG or PNG format.
3. You can organize photos into subdirectories if needed (e.g., `family`, `friends`).
4. Ensure to name the photos clearly to reflect their content.

## Deployment Information
1. Ensure that you have the latest version of the code from the repository.
2. Use a web server to host the website. You can use platforms like GitHub Pages, Vercel, or Netlify to deploy.
3. Follow the specific instructions on the platform you choose for deployment.
4. After deploying, test the website by accessing the provided URL.
